# 🛒 Fake Review Detection & Sentiment Analysis (NLP Project)

## 📌 Overview
This project analyzes e-commerce product reviews using Natural Language Processing (NLP) techniques.

It includes:
- Sentiment Analysis (VADER)
- Aspect-Based Sentiment Analysis (ABSA)
- Fake Review Detection using Suspicion Scoring
- Data Visualization Dashboard

---

## 🚀 Features

✔ Sentiment Classification (Positive, Negative, Neutral)  
✔ Aspect-Based Analysis (Delivery, Price, Build)  
✔ Fake Review Detection Model  
✔ Interactive Visualizations
